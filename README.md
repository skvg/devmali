# Devmali Heritage Website

A modern, responsive website showcasing the rich cultural heritage of Devmali village and the spiritual significance of Lord Devnarayan. Built with Next.js 14+, TypeScript, and Tailwind CSS.

## Features

- **Modern Design**: Responsive design with Rajasthani-inspired color palette
- **Multilingual Support**: English, Hindi, and Rajasthani languages
- **Cultural Heritage**: Showcase of village traditions and 1100-year history
- **Spiritual Content**: Lord Devnarayan teachings and devotional materials
- **Virtual Tour**: Immersive 360-degree village experience
- **Tourist Information**: Comprehensive travel and accommodation details
- **Community Stories**: Authentic testimonials and village resident stories

## Technology Stack

- **Framework**: Next.js 14+ with App Router
- **Language**: TypeScript with strict mode
- **Styling**: Tailwind CSS with custom Rajasthani theme
- **Fonts**: Inter, Noto Sans Devanagari, Playfair Display
- **Deployment**: AWS S3 static hosting with CloudFront CDN

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd devmali-heritage-website
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Color Palette

The website uses a Rajasthani-inspired color palette:

- **Saffron Orange** (#FF6B35) - Spiritual significance and warmth
- **Deep Maroon** (#8B0000) - Traditional Rajasthani textiles
- **Golden Yellow** (#FFD700) - Temple architecture and prosperity
- **Earth Brown** (#8B4513) - Mud house architecture
- **Sage Green** (#87A96B) - Aravalli hills and nature
- **Cream White** (#FFF8DC) - Purity and simplicity
- **Charcoal Gray** (#36454F) - Text and modern contrast
