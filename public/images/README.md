## ğŸ–¼ï¸ Image Guidelines

### **Image Specifications**

#### **High-Resolution Images (HD)**
- **Format**: JPEG (.jpg) preferred for photographs
- **Resolution**: Minimum 1920x1080px, up to 4K (3840x2160px)
- **Quality**: 85-95% JPEG quality
- **File Size**: 500KB - 2MB per image
- **Use Cases**: Virtual darshan, detailed heritage photos, hero backgrounds

#### **Thumbnail Images**
- **Format**: JPEG (.jpg) or WebP (.webp)
- **Resolution**: 400x300px to 800x600px
- **Quality**: 75-85% JPEG quality
- **File Size**: 50KB - 200KB per image
- **Use Cases**: Gallery previews, card thumbnails, quick loading

#### **Background Images**
- **Format**: JPEG (.jpg) for photographs, PNG (.png) for graphics
- **Resolution**: 1920x1080px minimum, 2560x1440px preferred
- **Quality**: 90-95% JPEG quality
- **File Size**: 300KB - 1MB per image
- **Use Cases**: Hero sections, page backgrounds, overlay images

#### **Profile Images (Testimonials)**
- **Format**: JPEG (.jpg) or PNG (.png)
- **Resolution**: 300x300px to 500x500px (square aspect ratio)
- **Quality**: 80-90% JPEG quality
- **File Size**: 30KB - 150KB per image
- **Use Cases**: Visitor testimonials, team photos

### **360-Degree Panoramic Images**
- **Format**: JPEG (.jpg)
- **Resolution**: 4096x2048px to 8192x4096px (2:1 aspect ratio)
- **Quality**: 90-95% JPEG quality
- **File Size**: 2MB - 8MB per image
- **Use Cases**: Virtual tours, immersive experiences

## ğŸ¥ Video Guidelines

### **Background Videos**
- **Format**: MP4 (.mp4) with H.264 codec
- **Resolution**: 1920x1080px (Full HD) minimum
- **Duration**: 30 seconds to 2 minutes (looping)
- **File Size**: 5MB - 20MB per video
- **Frame Rate**: 24-30 fps
- **Audio**: Optional, keep volume low if included

## ğŸµ Audio Guidelines

### **Spiritual Audio Content**
- **Format**: MP3 (.mp3) preferred
- **Quality**: 128-192 kbps
- **Duration**: 2-20 minutes depending on content type
- **File Size**: 2MB - 30MB per file
- **Use Cases**: Prayers, mantras, teachings, stories

### **Ambient Audio**
- **Format**: MP3 (.mp3)
- **Quality**: 128 kbps (sufficient for ambient sounds)
- **Duration**: 15-25 minutes (for looping)
- **File Size**: 15MB - 35MB per file
- **Use Cases**: Temple atmosphere, nature sounds, meditation

### **Tour Narration**
- **Format**: MP3 (.mp3)
- **Quality**: 192 kbps (clear speech)
- **Duration**: 2-10 minutes per location
- **File Size**: 3MB - 15MB per file
- **Languages**: English, Hindi, Rajasthani versions

## ğŸ“„ Document Guidelines

### **Downloadable PDFs**
- **Format**: PDF (.pdf)
- **Quality**: High-resolution text, optimized for reading
- **File Size**: 1MB - 15MB depending on content
- **Use Cases**: Prayer books, calendars, teachings, stories

### **Audio Collections**
- **Format**: ZIP (.zip) archives containing multiple MP3 files
- **File Size**: 20MB - 50MB per collection
- **Use Cases**: Complete mantra collections, prayer compilations

## ğŸš€ Optimization Tips

### **Before Adding Images:**

1. **Compress Images**: Use tools like TinyPNG, ImageOptim, or Squoosh
2. **Choose Right Format**: 
   - JPEG for photographs
   - PNG for graphics with transparency
   - WebP for modern browsers (with JPEG fallback)
3. **Resize Appropriately**: Don't upload larger images than needed
4. **Add Alt Text**: Ensure all images have descriptive alt text in code

### **Performance Considerations:**

- **Lazy Loading**: Images load as users scroll (already implemented)
- **Progressive JPEG**: Use progressive encoding for large images
- **Responsive Images**: Provide multiple sizes for different screen sizes
- **CDN Delivery**: Images will be served via CloudFront CDN

### **Cultural Sensitivity:**

- **Respectful Content**: Ensure all religious and cultural images are respectful
- **Authentic Representation**: Use genuine photos from Devmali village
- **Permission**: Obtain proper permissions for photos of people
- **Quality Standards**: Maintain high visual standards for spiritual content

## ğŸ“ File Naming Conventions

### **Naming Rules:**
- Use lowercase letters only
- Use hyphens (-) instead of spaces or underscores
- Be descriptive but concise
- Include size indicator for different versions

### **Examples:**
- `main-deity-hd.jpg` (high-resolution version)
- `main-deity-thumb.jpg` (thumbnail version)
- `mud-house-exterior.jpg` (descriptive name)
- `festival-celebration.jpg` (clear purpose)
